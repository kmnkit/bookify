### Components
1. **Top Navigation Bar**
   - Logo + menu toggle
   - Global search (desktop: expanded, mobile: icon)
   - User avatar

2. **Side Navigation (Desktop)**
   - Home, Search, Library, Profile icons
   - Collapsible to icon-only mode

3. **Content Sections**
   - Section headers with "See All" link
   - Horizontal scrollable book carousels
   - Book cards with cover, title, author, like button

4. **Bottom Navigation (Mobile)**
   - 4 main items: Home, Search, Library, Profile
   - Active state highlighted

### Design Notes
- Horizontal scroll for book rows (mobile)
- Grid layout for desktop (3-5 columns)
- Lazy loading for below-fold content
- Pull-to-refresh on mobile
- Skeleton loaders during data fetch

## Accessibility Annotations

All wireframes should be implemented with:
- Semantic HTML5 elements
- ARIA labels for all interactive elements
- Keyboard navigation support
- Focus indicators (visible outlines)
- Screen reader announcements for state changes
- Sufficient color contrast (4.5:1 for text)
- Touch targets minimum 44x44px
- Alternative text for all images
